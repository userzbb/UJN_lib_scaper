# UJN 图书馆自动预约签到工具

## 功能

- 自动预约指定座位
- 自动签到
- 查看预约列表
- 查看可用阅览室

## 安装依赖

```bash
uv sync
uv run playwright install chromium
```

## 配置文件

编辑 `config.json`:

```json
{
    "username": "202331223125",      // 你的学号
    "password": "080518",            // 你的密码
    "booking": {
        "room_id": 19,               // 阅览室ID
        "room_name": "第三阅览室北区", // 阅览室名称(仅显示用)
        "seat_id": 9418,             // 座位ID (可选,不填则自动查找)
        "seat_num": "260",            // 座位号 (用于自动查找座位ID)
        "date": "2026-03-26",        // 预约日期
        "start_time": "09:00",       // 开始时间
        "end_time": "12:00"          // 结束时间
    },
    "auto_checkin": true             // 预约成功后自动签到
}
```

## 使用方法

### 1. 查看可用阅览室

```bash
uv run python auto_reserve.py rooms
```

输出示例:
```
可用阅览室:
------------------------------------------------------
ID     名称                 校区     
------------------------------------------------------
8      第五阅览室北区       西校区   
17     第三阅览室北区       西校区   
19     第三阅览室南区       西校区   
...
```

### 2. 预约座位

```bash
uv run python auto_reserve.py book
```

预约成功示例:
```
预约信息:
  阅览室: 19 (第三阅览室北区)
  日期: 2026-03-26
  座位: 260 (ID: 9418)
  时间: 09:00 - 12:00
正在预约...
预约成功!
  预约号: 3125-xxx-x
  位置: 西校区4层407室区第三阅览室北区，座位号260
  时间: 09 : 00 - 12 : 00
  ID: 12345678

自动执行签到...
签到成功!
```

### 3. 命令行指定参数

可以不修改配置文件，直接在命令行指定参数:

```bash
# 指定日期预约
uv run python auto_reserve.py book --date 2026-03-27 --room 19 --seat 260 --start 14:00 --end 18:00

# 指定座位ID (跳过自动查找)
uv run python auto_reserve.py book --room 19 --seat 260 --date 2026-03-27 --id 9418
```

### 4. 签到

```bash
# 自动签到第一个可签到的预约
uv run python auto_reserve.py checkin

# 指定预约ID签到
uv run python auto_reserve.py checkin --id 12345678
```

### 5. 查看我的预约

```bash
uv run python auto_reserve.py list
```

输出示例:

## 快速开始

1. **修改配置文件** `config.json`，填入你的学号和密码

2. **查看可用阅览室**:
   ```bash
   uv run python auto_reserve.py rooms
   ```

3. **修改配置文件的 `room_id` 和 `seat_num`**

4. **预约**:
   ```bash
   uv run python auto_reserve.py book
   ```

## 常见问题

### Q: 如何知道座位号?

在配置文件中设置 `seat_num` (如 "260")，程序会自动在指定阅览室查找对应座位。

### Q: 如何知道阅览室ID?

运行 `uv run python auto_reserve.py rooms` 查看所有阅览室及其ID。

### Q: 预约失败?

可能原因:
- 座位已被预约
- 不在可预约时间内 (7:00-22:00)
- 预约时间超出允许范围

### Q: 签到失败?

签到需要在预约开始时间 **45分钟内** 完成，否则预约会自动取消。
